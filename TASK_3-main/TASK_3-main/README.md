# 📊 Deep-Dive Analysis & Interactive Dashboard

## 🚀 Project Overview
This project focuses on solving business problems using **data analysis and interactive dashboarding**.

The dashboard provides insights into:
- Revenue trends
- Customer behavior
- Regional performance
- Business funnel analysis

---

## 🎯 Objectives
- Define and track key KPIs
- Perform deep-dive analysis
- Build an interactive dashboard

---

## 📌 Key KPIs
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value (AOV)

---

## 🔍 Analysis Performed

### 1. 📈 Sales Trend Analysis
- Visualized revenue over time

### 2. 🌍 Region-wise Analysis
- Compared performance across regions

### 3. 👤 Customer Analysis
- Identified top contributing customers

### 4. 📅 Cohort Analysis
- Monthly customer retention overview

### 5. 🔻 Funnel Analysis
- User journey (Visited → Cart → Purchase)

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
